There are 4 of request response pairs in total. 1 of them is due to TCP connection establishment, one is for HTML page request, and 1 is for object request. Every second request response pair is for the TCP connection establishment. Legs of the triangles corresponding to the server responses on HTML page and object requests are equal to the other legs of these triangles representing the respective client requests. Total round trip time equals 4.
